Votre site est en ligne et tout est déjà configuré. Il vous reste seulement quelques minutes pour devenir le seul propriétaire de vos accès — votre site reste en ligne pendant ce temps.

Bonjour,

Voici vos accès GitHub (le code de votre site) et Vercel (la mise en ligne). Suivez ces 3 étapes à votre rythme.

PlateformesGitHub + Vercel
Identifiant (e-mail)[ votre adresse ]
Mot de passe provisoirevia lien sécurisé →

  1. Récupérez votre mot de passe provisoire

    Nous vous avons envoyé un lien sécurisé (séparément du présent document). Ouvrez-le et copiez le mot de passe. ⚠️ Ce lien est à usage unique et expire : ouvrez-le au moment de vous connecter.

  2. Validez votre e-mail (dans votre boîte mail)

    Vous avez reçu un e-mail de GitHub à votre adresse : ouvrez-le et cliquez sur « Verify email address ». C'est tout.

  3. Connectez-vous & sécurisez votre compte

    Allez sur github.com, connectez-vous avec votre e-mail + le mot de passe provisoire. Puis Settings → Password pour mettre votre mot de passe, et activez la 2FA (Settings → Password and authentication).

→

Votre nom de domaine (votre adresse web) est traité dans un document dédié : « Connecter votre nom de domaine ». Rien à faire ici pour cela.

✓

Déjà fait par nos soins : votre adresse e-mail principale, la configuration du projet et la mise en ligne. Vous n'avez aucun réglage technique à toucher.

✓

Pour Vercel : rien à installer. Sur vercel.com, cliquez « Continue with GitHub » — votre projet apparaît automatiquement.
